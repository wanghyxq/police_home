# 修改理发师工作台列表样式

根据您的要求，我们将更新 `pages/barber/dashboard.vue` 中列表项的样式结构，使其与提供的 HTML 模板一致，并仅显示完整的手机号（移除用户名）。

## 1. 样式调整

我们将把当前的 `<view>` 结构修改为：

*   **布局**: 保持 `flex justify-between items-center`。
*   **左侧内容**:
    *   第一行: 时间 (`font-mono text-xl font-bold text-police mb-1`)。
    *   第二行: 日期 (`text-xs text-gray-500`，带日历图标)。
    *   第三行: 用户信息 (`text-xs text-gray-500 mt-1`)。**关键修改**：只显示完整手机号，格式为 `(尾号xxxx)` 或直接显示完整号。根据您的最新要求“只显示完整手机号”，我们将直接显示 `item.userPhone`。
*   **右侧内容**:
    *   状态标签 (`text-xs mr-2`)。
    *   右箭头图标。

## 2. 代码实现

我们将替换 `v-for` 循环内部的模板代码。

### 新的模板结构

```html
<view v-for="(item, index) in taskList" :key="index" @click="goDetail(item.id)"
  class="bg-white p-4 rounded-xl shadow-sm border border-gray-100 flex justify-between items-center active:bg-gray-50 cursor-pointer">
  <view>
    <view class="font-mono text-xl font-bold text-police mb-1">{{ item.time }}</view>
    <view class="text-xs text-gray-500 flex items-center">
        <i class="fa-regular fa-calendar mr-1"></i>{{ item.date }}
    </view>
    <view class="text-xs text-gray-500 mt-1">
        {{ item.userPhone }}
    </view>
  </view>
  <view class="text-right flex items-center">
    <view class="text-xs mr-2">
      <text v-if="item.status === '0' || item.status === 'booked'" class="text-green-600 font-bold">待服务</text>
      <text v-else-if="item.status === '1' || item.status === 'pending_eval'" class="text-orange-500">待评价</text>
      <text v-else class="text-gray-400">已完成</text>
    </view>
    <i class="fa-solid fa-chevron-right text-gray-300 text-xs"></i>
  </view>
</view>
```

**注意**:
1.  我将同时兼容后端返回的状态码 `'0', '1'` 和前端可能使用的旧状态码 `'booked'` 等。
2.  `active:bg-gray-50` 是 Tailwind 类，用于点击反馈。
